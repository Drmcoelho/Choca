# Figuras · Atlas farmacológico M28

Figuras editoriais em SVG para o atlas farmacológico do braço **PERFUNDE · CHOCA**.

Arquivos:

- `atlas-receptores-hemodinamicos.svg` — mapa receptor → efeito inicial → fármaco representativo.
- `dobutamina-fluxo-pressao.svg` — dobutamina como droga de integração fluxo × pressão.
- `vasopressores-comparados.svg` — matriz comparativa de vasopressores por fenótipo.
- `combinacoes-hemodinamicas.svg` — combinações por fenótipo, com foco em dobutamina.
- `choque-septico-surviving.svg` — aba Surviving do choque séptico.

Convenção: imagens SVG leves, versionáveis e editáveis, prontas para embed direto nos HTMLs via `<img>` ou `<figure>`.
